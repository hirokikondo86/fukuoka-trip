## 個別の事前徴収金額

交通費に関しては、関西勢と関東勢別々で徴収致します。ここでは関西勢の内訳が記述されていますので、関東勢は各々で調整して下さい。

宿泊費 (Airbnb) に関しては、1 人 (別途ホテルに宿泊) を除いた金額で徴収致します。

- 交通費 (関西勢) - 140,130 円 / 9 人 = 15,570 円
- 宿泊費 (Airbnb 勢) - 97,776 円 / 11 人 = 8,888 円
